# Tandem Product Overview

## Core loop

1. Choose a folder to work in
2. Ask for help in chat (or run a starter pack)
3. Review proposed changes
4. Apply changes with explicit approval
5. Undo when needed (Git-backed history)

## Key features

- Folder-wide context (not just one file)
- Permissioned file operations
- Execution planning (Plan Mode)
- Iterative agent loop (Ralph Mode)
- Offline starter packs and skills

## Primary personas

- **Writer**: improve drafts, consistency, tone
- **Researcher**: synthesize sources, identify conflicts, build briefs
- **Ops/Admin**: organize docs, create checklists, summarize threads
