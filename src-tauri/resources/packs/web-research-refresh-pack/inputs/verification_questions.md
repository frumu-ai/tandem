## Research Principles

### Source Priority

1. Official documentation (docs.github.com, taurapps.org, etc.)
2. Official vendor blogs and announcements
3. GitHub repositories (official)
4. Authoritative industry standards

### What NOT to Use

- Paywalled analyst reports
- Third-party blog posts (unless official source)
- Reddit or community forum posts as primary sources
- Wikipedia as a primary source

### Citation Requirements

- Every fact must have a source URL
- Record access date for each source
- Note if source is official or unofficial
